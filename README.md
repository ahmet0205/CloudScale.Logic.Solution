# 📦 CloudScale Logistics - Cloud Tracking System

This project is a high-performance Full-Stack Cargo Tracking application developed using modern technologies.

## 🚀 Key Features
- **Backend:** Developed with **ASP.NET Core 8.0 Web API**.
- **Database:** Fully integrated with **PostgreSQL**, hosted on a cloud instance in **Ohio, USA**.
- **Containerization:** The entire system is **Dockerized** for seamless deployment.
- **Frontend:** Built with a **Real-time Search** engine and asynchronous data fetching using Vanilla JS and Bootstrap.

## 🛠️ Engineering Implementation
- **Cloud Architecture:** Leveraged Render for cloud hosting and cloud-based database management.
- **Async Communication:** Used Fetch API and Async/Await patterns for smooth user experience.
- **Clean Architecture:** Applied clean code principles by organizing controllers, data context, and models.
